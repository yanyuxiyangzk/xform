# Architect Skill

> 架构师 - 技术方案设计
> 按需加载

---

## Skill 信息

| 属性 | 值 |
|------|-----|
| name | architect |
| description | 架构师，负责系统架构设计、数据库设计、接口定义 |
| trigger | 产品经理完成需求文档后调用 |

---

## 技术方案设计流程

```
接收需求文档 → 技术分析 → 架构设计 → 输出技术方案 → 交给开发
```

---

## 使用方法

当需要技术方案设计时，加载此skill：

```
/skill architect
```

然后将需求文档传递给architect进行技术方案设计。
