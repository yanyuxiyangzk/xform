## 重要性评分

| 分数 | 标准 | 示例 |
|------|------|------|
| 9-10 | 业务关键、法律义务 | 投资决策、核心技术选型 |
| 7-8 | 核心决策、重要关系 | 架构决策、关键人员 |
| 5-6 | 有用参考信息 | 技术笔记、会议摘要 |
| 3-4 | 一般记录 | 每日日志、想法 |
| 1-2 | 临时/一次性 | 快速笔记、即将过期的信息 |

## 自动触发规则

### 层级提升

```
情节 → 工作：access_count > 5 AND age > 7天
工作 → 长期：importance >= 7 OR access_count > 10
```

### 归档规则

```
情节记忆：30天后自动归档或删除
工作记忆：30天后评估是否提升到长期
长期记忆：永久保留，除非手动删除
```

3. 时间接近度
