## 1. 项目背景

### 1.1 项目概述
- 项目名称：RuoYi-Cloud-Nocode
- 项目类型：零代码微服务平台
- 目标用户：企业开发团队、业务人员

### 1.2 现有能力
- 基础框架已搭建（Java 17 + Spring Cloud + Vue3）
- Liquor动态编译功能已完成
- 用户模块已完成

### 1.3 本次目标
开发零代码核心功能，实现：
1. 表单设计器 - 可视化配置业务表单
2. 流程引擎 - 业务流程配置
3. 代码生成器 - 一键生成CRUD代码

---

## 2. 功能需求

### 2.1 表单设计器

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 拖拽组件 | 拖拽表单组件到画布 | P0 |
| 属性配置 | 配置组件属性 | P0 |
| 数据绑定 | 绑定表单字段到数据模型 | P0 |
| 预览 | 实时预览表单效果 | P1 |
| 保存/发布 | 保存表单配置并发布 | P0 |

**表单组件**：
- 文本输入 (text)
- 数字输入 (number)
- 下拉选择 (select)
- 日期选择 (date)
- 开关 (switch)
- 多行文本 (textarea)

### 2.2 流程引擎

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 流程设计器 | 可视化设计审批流程 | P0 |
| 节点配置 | 配置开始、中间、结束节点 | P0 |
| 审批人设置 | 设置审批人/审批组 | P0 |
| 条件分支 | 支持条件判断分支 | P1 |
| 流程执行 | 执行提交流程 | P1 |

### 2.3 代码生成器

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 表单配置解析 | 解析表单设计器配置 | P0 |
| 后端代码生成 | 生成Entity、Mapper、Service、Controller | P0 |
| 前端代码生成 | 生成Vue页面和API | P0 |
| 一键部署 | 自动部署到开发环境 | P2 |

---

## 3. 技术架构

### 3.1 技术栈
- 后端：Java 17 + Spring Boot 3.2 + MyBatis-Plus
- 前端：Vue3 + Element Plus + TypeScript
- 动态编译：Liquor

### 3.2 模块结构
```
ruoyi-nocode/
├── ruoyi-nocode-generator/    # 代码生成模块
├── ruoyi-nocode-form/         # 表单设计模块
├── ruoyi-nocode-workflow/     # 流程引擎模块
```
