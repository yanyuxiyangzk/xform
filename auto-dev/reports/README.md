# 报告输出目录

> 本目录存储定期生成的开发报告

---

## 目录结构

```
reports/
├── daily/                     # 每日报告
│   └── 2026-04-02.md
│
├── weekly/                    # 周报告
│   └── 2026-W14.md
│
└── README.md                  # 本文件
```

---

## 日报模板

```markdown
# 📅 YYYY-MM-DD 开发日报

## 今日完成
- [x] TASK-ID: 任务描述

## 明日计划
- [ ] TASK-ID: 任务描述

## 问题与风险
- 无 | 问题描述

## 代码统计
- 新增: X 行
- 修改: X 行
- 删除: X 行

## 消耗统计
- Token: X.XM
- 循环次数: X
```

---

## 周报模板

```markdown
# 📊 Week X (YYYY-MM-DD ~ YYYY-MM-DD) 周报

## 本周完成
- [x] TASK-ID: 任务描述

## 项目进度
- 总体进度: XX%
- 完成功能: X 个
- 代码变更: X 行

## 下周计划
- [ ] TASK-ID: 任务描述

## 问题汇总
- 问题1及解决方案
- 问题2及解决方案

## 里程碑
- 已达成: 里程碑1
- 进行中: 里程碑2
- 待开始: 里程碑3
```
